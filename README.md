### Step 3: Create setup.ps1

Create `skills/setup.ps1`:

```powershell
# Skills Setup Script
$ScriptDir = Split-Path -Parent $MyInvocation.MyCommand.Path
$RepoRoot = Split-Path -Parent $ScriptDir

Write-Host "AI Skills Setup" -ForegroundColor Cyan

$agentsMd = Join-Path $RepoRoot "AGENTS.md"
if (-not (Test-Path $agentsMd)) {
    Write-Host "ERROR: AGENTS.md not found!" -ForegroundColor Red
    exit 1
}

# Claude
$claudeDir = Join-Path $RepoRoot ".claude"
New-Item -ItemType Directory -Force -Path $claudeDir | Out-Null
$skillsTarget = Join-Path $claudeDir "skills"
if (Test-Path $skillsTarget) { Remove-Item -Recurse -Force $skillsTarget }
Copy-Item -Recurse $ScriptDir $skillsTarget
Copy-Item $agentsMd (Join-Path $claudeDir "CLAUDE.md")
Write-Host "+ .claude/" -ForegroundColor Green

# Gemini
$geminiDir = Join-Path $RepoRoot ".gemini"
New-Item -ItemType Directory -Force -Path $geminiDir | Out-Null
$skillsTarget = Join-Path $geminiDir "skills"
if (Test-Path $skillsTarget) { Remove-Item -Recurse -Force $skillsTarget }
Copy-Item -Recurse $ScriptDir $skillsTarget
Copy-Item $agentsMd (Join-Path $geminiDir "GEMINI.md")
Write-Host "+ .gemini/" -ForegroundColor Green

# GitHub Copilot
$githubDir = Join-Path $RepoRoot ".github"
New-Item -ItemType Directory -Force -Path $githubDir | Out-Null
Copy-Item $agentsMd (Join-Path $githubDir "copilot-instructions.md")
Write-Host "+ .github/" -ForegroundColor Green

Write-Host "Done! Restart your AI assistant." -ForegroundColor Green
```

## âš¡ Lessons Learned

### âœ… DO

- **One place for skills**: Keep all skills in `root/skills/`
- **Auto-invoke table**: Without this table, models ignore skills
- **Centralized setup.ps1**: One script that syncs everything
- **AGENTS.md at root**: The master file always at project root

### âŒ DON'T

- **Don't duplicate skills**: Don't have skills in multiple folders
- **Don't forget Auto-invoke**: It's critical for models to use skills
- **Don't create AGENTS.md per subfolder**: One master AGENTS.md is enough
